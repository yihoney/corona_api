# corona_api
- 관련 api: https://www.data.go.kr/data/15043379/openapi.do
- 대륙별 페이징 처리 구현
- 해당 국가를 클릭하면 2022.09.01 ~ 09.31까지의 현황을 조회합니다.
- 팝업 또는 새로운 창으로 이동합니다.
- chart : google region chart를 이용해서 구현합니다.
